# GameScreenshots 卸载说明

## 适用插件

| 插件 | ID | 当前版本 |
|---|---|---|
| 游戏截图 | `Game_Screenshots_5139A212-C04C-419F-A534-71DA19581A63` | `1.9` |

## 卸载步骤

| 步骤 | 操作 |
|---|---|
| 1 | 先完全退出 Playnite。 |
| 2 | 删除目录 `C:\Users\Administrator\AppData\Roaming\Playnite\Extensions\Game_Screenshots_5139A212-C04C-419F-A534-71DA19581A63\`。 |
| 3 | 如果你不想保留截图数据，再删除插件用户数据目录里的截图文件。默认就在上面这个插件目录下。 |
| 4 | 重新启动 Playnite，确认侧边栏里的 `画廊` 已消失。 |

## 注意

| 项目 | 说明 |
|---|---|
| 会删除什么 | 插件本体会被删。 |
| 可能一并删掉什么 | 如果你把截图也一起删了，之前保存的截图文件会一起没掉。 |
| 不会删什么 | 不会改动 Playnite 主程序，也不会影响别的插件。 |
