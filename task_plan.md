# 任务计划

目标：新增 Switch 智能导入插件，解决 Switch 本体、补丁、DLC 和重复包误导入问题，并支持待确认列表与可选全量元数据刷新。

| 实现扫描、分类、归并和待确认缓存 | 未开始 |
| 实现导入与元数据全量刷新 | 未开始 |
| 运行测试、构建和打包 | 未开始 |

## 边界
| 项目 | 说明 |
|---|---|
| 支持范围 | 只处理 Switch 包：`.nsp`、`.nsz`、`.xci`、`.xcz` |
| 识别优先级 | 文件名标记、Title ID、版本号优先，目录结构只做辅助 |
| 不处理 | Playnite 原生导入窗口替换、非 Switch 文件、自动直接入库 |
| 元数据 | 来源为空时跳过；选 `Switch Local Metadata` 时全量刷新，不只补空字段 |
