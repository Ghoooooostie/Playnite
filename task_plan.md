# 任务计划

目标：在 Playnite 全屏首页游戏列表下方新增插件可填充的截图区域，并让 GameScreenshots 插件提供紧凑版截图内容。

## 步骤
| 步骤 | 状态 |
|---|---|
| 读取全屏首页主题和插件自定义元素机制 | 已完成 |
| 补充截图插件全屏首页控件测试 | 已完成 |
| 在默认全屏首页游戏列表下方加入截图插件插槽 | 已完成 |
| 实现 GameScreenshots 全屏紧凑截图控件 | 已完成 |
| 构建、测试和打包 GameScreenshots 1.4 | 已完成 |

## 边界
| 项目 | 说明 |
|---|---|
| 支持范围 | 默认全屏主题；GameScreenshots 插件；当前游戏最近 5 张截图 |
| 不处理 | 重新设计全屏首页、替换现有游戏回顾插槽、重做截图存储格式 |
| 注意 | 没有插件内容时截图区域自动隐藏；需要带本次主题改动的 Playnite 才能显示全屏首页区域 |
